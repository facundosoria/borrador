# XY1-S017 - What Counts as an Attack?

```
WHAT COUNTS AS AN ATTACK?
Every attack has a cost and a name, and it might have a base damage and damage text. Most of the text on a Pokémon describes its
attack, even if it doesn’t do any damage.
For example, Panpour’s Collect does no damage, but it’s still an attack! Anything else is called an Ability or something else. For
instance, Dusknoir’s Sinister Hand Ability moves damage counters around, but it doesn’t count as an attack.
Koffing’s Smokescreen affects Collect, but not Sinister Hand.
```
