# XY1-S006 - 3 Card Types

```
You’ll find 3 different types of cards in the Pokémon TCG:
Pokémon (Basic, Stage 1, Stage 2)
Energy (required to attack, any kind can be used for colorless)
Trainer (Item, Supporter, Stadium)
Each player has his or her own discard pile.
When a Pokémon is Knocked Out, it and any cards attached are sent to discard.
```
