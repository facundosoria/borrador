# XY1-S022 - Name Counting

```
When you are building a deck, you can have only 4 copies of a card with the same name in it, except for basic Energy cards.
Symbols at the end of a Pokémon’s name are part of a Pokémon’s name.
An owner in a Pokémon’s name is part of the Pokémon’s name.
However, δ (Delta Species) is not part of a Pokémon’s name.
```
