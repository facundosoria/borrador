# Deck Construction

## Resumen
Reglas base para construir mazos segun el TPI.

## Reglas exactas
- El mazo debe tener exactamente 60 cartas.
- Maximo 4 cartas con el mismo nombre (excepto Energia Basica).
- Energia Especial: maximo 4 copias en el mazo.
- Maximo 1 carta ACE SPEC total en el mazo.
- Minimo 1 Pokemon Basico.
