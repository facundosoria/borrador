# Pokemon-EX Rules

## Resumen
Reglas especiales para Pokemon-EX.

## Reglas exactas
- EX es parte del nombre del Pokemon-EX.
- Si un Pokemon-EX es KO, el rival toma 2 premios.
- Por lo demas, se juega como Pokemon normal.
