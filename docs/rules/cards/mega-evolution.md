# Mega Evolution Rules

## Resumen
Reglas especiales para Mega Evolucion.

## Reglas exactas
- La M es parte del nombre.
- Mega Evolucion evoluciona desde Pokemon-EX especifico.
- Cuenta como carta de Evolucion y sigue reglas normales.
- Si una Mega Evolucion es KO, el rival toma 2 premios.
- Cuando un Pokemon se convierte en Mega Evolucion, el turno termina.
