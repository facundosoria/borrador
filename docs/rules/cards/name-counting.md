# Name Counting

## Resumen
Reglas para contar copias por nombre.

## Reglas exactas
- Simbolos al final del nombre cuentan como parte del nombre.
- El nombre de propietario cuenta como parte del nombre.
- Delta Species no cuenta como parte del nombre.
- Para mazos, maximo 4 cartas por nombre (excepto Energia Basica).
