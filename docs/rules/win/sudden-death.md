# Sudden Death

## Resumen
Regla de muerte subita cuando ambos ganan a la vez.

## Reglas exactas
- Si ambos jugadores ganan simultaneamente, jugar Muerte Subita.
- En Muerte Subita, se juega una nueva partida con 1 carta de Premio por jugador.
- Todo lo demas se juega como una partida normal.
- Si termina otra vez en Muerte Subita, repetir hasta que haya ganador.
