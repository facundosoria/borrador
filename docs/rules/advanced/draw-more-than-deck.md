# Draw More Cards Than Deck

## Resumen
Regla cuando un efecto pide robar/ver mas cartas de las que quedan.

## Reglas exactas
- Si un efecto pide robar o mirar mas cartas de las que quedan, usar solo las disponibles.
- No se pierde por efecto de carta que pide robar mas; se pierde solo si no se puede robar al inicio del turno.
