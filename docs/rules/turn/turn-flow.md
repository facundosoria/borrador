# Turn Flow

## Resumen
Flujo general de un turno y restricciones.

## Reglas exactas
- El turno tiene tres partes: robar carta, acciones en cualquier orden, atacar y terminar.
- Una vez que atacas, el turno termina y no puedes volver a acciones previas.
