# Up To vs Any Number

## Resumen
Interpretacion de limites en efectos de cartas.

## Reglas exactas
- "Up to" permite elegir cualquier cantidad entre 0 y el maximo indicado (inclusive).
- "Any number" no impone limite superior mas alla de las reglas del juego.
