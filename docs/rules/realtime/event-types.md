# Realtime Event Types

## Resumen
Eventos que deben emitirse para sincronizacion en tiempo real.

## Reglas exactas
- Emitir eventos por turno, ataque, KO, premio, condicion especial y fin de partida.
- Sincronizar estado tras cada accion valida.
