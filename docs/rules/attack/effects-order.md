# Attack Effects Order

## Resumen
Orden de efectos durante un ataque.

## Reglas exactas
- Si el ataque es complejo, seguir el orden: decisiones -> requisitos -> efectos que alteran/cancelan -> efectos antes del dano -> dano -> otros efectos.
- Los efectos que cambian dano se aplican antes de colocar contadores.
