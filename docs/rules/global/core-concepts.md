# Core Concepts

## Resumen
Conceptos base necesarios para interpretar cualquier regla de juego.

## Reglas exactas
- Tipos de carta: Pokemon (Basico, Stage 1, Stage 2), Energia, Entrenador (Item, Supporter, Stadium).
- Los Energies deben coincidir con el costo de ataque, pero el simbolo incoloro puede ser cualquier tipo.
- Cada jugador tiene su propia pila de descartes.
- Si un Pokemon es KO, el Pokemon y cartas adjuntas van a descarte.
- Zonas: mano, premios, mazo, descarte, banca, activo, en juego.
- La mano del rival es oculta salvo efecto de carta.

## Depende de
- Ninguno.
