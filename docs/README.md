# Docs

## Estructura
- `requirements/`: RF/RNF y trazabilidad.
- `rules/`: reglas del juego modulares con dependencias.
- `api/`: OpenAPI y eventos.
- `architecture/`: diagramas y limites.
- `testing/`: estrategia de pruebas.
- `delivery/`: checklist final.

## Regla operativa
Para implementar una accion, leer `docs/rules/index.md` y cargar solo los modulos listados.
